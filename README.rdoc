= growl-logger

http://github.com/jugyo/growl-logger

== DESCRIPTION:

Logger using Growl.

== FEATURES/PROBLEMS:

* You can output logs as growl notification so easily!

That's all.

== SYNOPSIS:

  require 'rubygems'
  require 'growl-logger'
  
  log = GrowlLogger.new :level => Logger::DEBUG
  
  log.debug('debug')
  log.info('info')
  log.warn('warn')
  log.error('error')
  log.fatal('fatal')
